Cloud-Based Residential HEMS Application

Project Overview

This cloud-based Home Energy Management System executes the MATLAB HEMS controller through GitHub Actions, reads forecasting data from the cloud, and displays the generated EMS results through this web dashboard.

Cloud Execution

The MATLAB controller is executed on a GitHub Actions cloud runner. The generated EMS decisions, summary files, and figures are updated automatically inside the GitHub repository.
